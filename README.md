# Van Jally Opon — Portfolio

Premium personal portfolio for a Senior Full Stack Engineer.

## Stack

- Next.js 16 + TypeScript
- Tailwind CSS 4
- Framer Motion
- Lucide React + React Icons

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Edit content

All profile copy lives in `src/data/profile.ts`:

- Experience, skills, projects, AI expertise
- Social links (LinkedIn / GitHub)
- Resume download (`public/resume/Van_Jally_Opon.pdf`)

## Scripts

- `npm run dev` — development server
- `npm run build` — production build
- `npm run start` — serve production build
- `npm run lint` — ESLint
